PlusXBit v1.0
Created by Corey Ehmke (www.idolhands.com)

---------------------------------------------------------------------

PlusXBit lets you set the execute bit on the selected file(s) from the Finder, without a trip to Terminal.

To install:

  1) Add to your apps folder
  2) Drag to the toolbar in a Finder window
  
To use:

  1) Select one or more files
  2) Click the PlusXBit icon in the toolbar
  3) That's it!
